## 🎯 Quick Start (3 Steps)

### Step 1: Add to Your HTML Pages
```html
<!-- Add to <head> -->
<link href="../styles/main.css" rel="stylesheet">

<!-- Add to <body> -->
<a href="#main-content" class="skip-nav">Skip to main content</a>

<!-- Add before </body> -->
<script src="../js/accessibility.js"></script>
```

### Step 2: Use Components
```html
<!-- Glass Card -->
<div class="glass-card p-8">
  <h2>Your Content</h2>
</div>

<!-- Button -->
<button class="btn-primary-gradient">Click Me</button>

<!-- Form -->
<div class="form-field">
  <label for="email" class="form-label">Email</label>
  <input type="email" id="email" class="form-input" required>
</div>
```

### Step 3: Show Notifications
```javascript
// Toast notification
window.toastManager.show('Success!', 'success');

// Announce to screen readers
window.accessibilityManager.announce('Loading complete');

// Open modal
window.modalManager.open('my-modal');
```

## 💻 How to Use Each Feature

### Toast Notifications
```javascript
// Success
window.toastManager.show('Changes saved!', 'success');

// Error
window.toastManager.show('Something went wrong', 'error');

// Warning
window.toastManager.show('Please review', 'warning');

// Info
window.toastManager.show('New update available', 'info');
```

### Modal Management
```javascript
// Open modal
window.modalManager.open('confirm-modal');

// Close modal
window.modalManager.close('confirm-modal');
```

### Accessibility Announcements
```javascript
// Announce to screen readers
window.accessibilityManager.announce('Loading complete');

// Announce with priority
window.accessibilityManager.announce('Error occurred', 'assertive');
```

### Theme Toggle
```javascript
// Theme toggle is automatic
// Just include accessibility.js
// Toggle button appears automatically
// Theme persists in localStorage
```

### Form Validation
```html
<div class="form-field">
    <label for="email" class="form-label">
        Email <span class="required">*</span>
    </label>
    <input 
        type="email" 
        id="email" 
        class="form-input" 
        required
        aria-required="true"
    >
    <div class="form-helper">We'll never share your email</div>
</div>
```
